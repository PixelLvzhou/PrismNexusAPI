## Implementation Notes

### 权限编码定义
```javascript
const PERMISSION_CODES = {
  NAV_MANAGE: { code: 'NAV_MANAGE', name: '导航管理权限' },
  ACCOUNT_MANAGE: { code: 'ACCOUNT_MANAGE', name: '账户管理权限' },
  USER_MANAGE: { code: 'USER_MANAGE', name: '用户管理权限' }
};
```

### 用户权限状态数据结构
```javascript
{
  permissions: [
    {
      code: 'NAV_MANAGE',
      name: '导航管理权限',
      granted: true,        // 是否已拥有
      granted_at: '2024-01-01T12:00:00Z',  // 授予时间
      granted_by: 1         // 授予人ID
    }
  ]
}
```

### 申请记录状态
- `pending` - 待审批
- `approved` - 已通过
- `rejected` - 已驳回
- `expired` - 已过期
- `cancelled` - 已取消
