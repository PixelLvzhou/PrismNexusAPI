```javascript
const fetchAmapWeatherForecast = async (cityCode) => {
  const amapKey = process.env.AMAP_KEY;
  if (!amapKey || amapKey === 'your_amap_key_here') {
    throw new Error('未配置高德地图API密钥');
  }

  const url = `https://restapi.amap.com/v3/weather/weatherInfo?city=${cityCode}&key=${amapKey}&extensions=all`;

  return new Promise((resolve, reject) => {
    const timeout = setTimeout(() => {
      reject(new Error('请求超时'));
    }, 10000);

    https.get(url, (response) => {
      clearTimeout(timeout);
      let data = '';
      response.on('data', (chunk) => data += chunk);
      response.on('end', () => {
        try {
          const result = JSON.parse(data);
          if (result.status === '1' && result.forecasts && result.forecasts.length > 0) {
            resolve(result.forecasts);
          } else {
            reject(new Error('获取天气预报失败: ' + (result.info || '未知错误')));
          }
        } catch (e) {
          reject(e);
        }
      });
      response.on('error', (err) => {
        clearTimeout(timeout);
        reject(err);
      });
    }).on('error', (err) => {
      clearTimeout(timeout);
      reject(err);
    });
  });
};
```

**新增天气预测接口**（预留）:
```typescript
export interface WeatherForecast {
  date: string;
  dayWeather: string;
  nightWeather: string;
  dayTemp: string;
  nightTemp: string;
  dayWind: string;
  nightWind: string;
}

export const getWeatherForecast = async (options?: {
  city?: string;
  lat?: number;
  lon?: number;
}): Promise<WeatherForecast[]> => {
  const params = new URLSearchParams();
  if (options?.city) params.append('city', options.city);
  if (options?.lat) params.append('lat', options.lat.toString());
  if (options?.lon) params.append('lon', options.lon.toString());

  const response = await fetch(
    `http://localhost:3000/api/weather/forecast?${params.toString()}`
  );

  if (!response.ok) {
    const errorData = await response.json();
    throw new Error(errorData.error || '获取天气预报失败');
  }

  return await response.json();
};
```

## 五、关键技术点

### 5.1 高德地图API使用

**地理编码API**:
- 接口: `https://restapi.amap.com/v3/geocode/geo`
- 参数: `address`（地址）、`key`（API密钥）
- 返回: `adcode`（行政区划代码）、`formatted_address`（格式化地址）

**实时天气API**:
- 接口: `https://restapi.amap.com/v3/weather/weatherInfo`
- 参数: `city`（城市代码）、`key`（API密钥）、`extensions=base`
- 返回: 实时天气数据，包含温度、湿度、风向、风力等

**天气预测API**:
- 接口: `https://restapi.amap.com/v3/weather/weatherInfo`
- 参数: `city`（城市代码）、`key`（API密钥）、`extensions=all`
- 返回: 未来几天天气预报数据

### 5.2 城市名称精确到区/县

**实现方式**:
1. 使用高德地理编码API获取 `formatted_address`
2. 解析地址字符串，提取省、市、区/县信息
3. 组合为精确的城市名称，如"北京市朝阳区"

**示例**:
```javascript
const formattedAddress = '北京市朝阳区望京街道';
// 解析后: city = '北京市朝阳区'
```

### 5.3 风向和风力展示

**风向**:
- 高德API返回风向字符串，如"东南风"
- 提取方向部分："东南"

**风力等级**:
- 高德API返回风力等级，如"3级"
- 同时计算风速（km/h）= 风力等级 × 5（近似值）

**展示格式**:
```
风向: 东南
风力: 3级
风速: 15km/h
```

## 六、风险与注意事项

### 6.1 高德API限制
- **调用频率**: 个人开发者每天最多调用3000次
- **解决方案**: 实现前端缓存，减少重复调用

### 6.2 浏览器定位权限
- **问题**: 用户可能拒绝定位权限
- **解决方案**: 提供手动选择城市的功能（后期扩展）

### 6.3 跨域问题
- **问题**: 前端调用后端API可能存在跨域
- **解决方案**: 后端已配置CORS中间件

### 6.4 数据更新频率
- **问题**: 高德天气数据更新频率有限
- **解决方案**: 前端每30分钟刷新一次，避免频繁调用
